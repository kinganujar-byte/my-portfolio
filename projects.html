Multi-Timezone Digital Clock

A real-time digital clock application that displays the current time across 12 major world timezones.

Features:

  • Real-time clock updates every second
  • Displays 12 major timezones (New York, London, Paris, Dubai, India, Singapore, Tokyo, Sydney, Los Angeles, Toronto, São Paulo, Hong Kong)
  • Shows local timezone information
  • Responsive design for all devices
  • Modern gradient styling with smooth animations

Technologies Used:

HTML5 CSS3 JavaScript Intl API
